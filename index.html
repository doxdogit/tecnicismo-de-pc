Opciones de Procesadores

Hay varios chips de AMD para elegir:

  1. Duron
  2. Sempron
  3. Athol
  4. Athol MP
  5. Opteron

AMD Chips

Chips Descripcion
Duron Duron salió al mercado en el verano de 2000 como una alternativa económica a los procesadores Athlon de AMD, Pentium III de Intel y Celeron de Intel Duron es compatible con Athlon, casi indistinguible, y funciona en la misma placa base en la mayoría de los casos Duron es una buena opción para un ordenador económico.
Sempron El Sempron es el último procesador de gama baja de AMD. Sustituye al procesador Duron y competirá con el procesador Celeron D de Intel para un ordenador económico, si la diferencia de precio entre Duron y Sempron es pequeña, elija Sempron
Athlon Este es el procesador de gama media de AMD.
Athlon MP Puede utilizarse en placas base compatibles con multiprocesadores
Athlon 64 Procesador de 64 bits de calidad de consumidor para sistemas de gama alta
Opteron Servidor/procesador de 64 bits de gama alta. Para servidores de gama alta

Procesador Intel

Hay varios chips de Intel para elegir:

  1. Celeron
  2. Celeron D
  3. Pentium 3
  4. Pentium 4
  5. Pentium D
  6. Xeon

Intel Chips

Chips Descripcion
Celeron Es la familia de chips económicos de Intel. Celeron es una buena opción para un ordenador económico, ya que ofrece rendimiento suficiente para un uso informático general
Celeron D la última incorporación de Intel a la familia Celeron, más rápido que el Celeron normal, pero todavía más barato.
Pentium 3 es el actual chip de gama media de Intel
Pentium 4 Puede utilizarse en placas base compatibles con multiprocesadores
Pentium D Es un chip de doble núcleo de gama alta. Contiene dos chips Pentium, es decir, dos procesadores en un solo chip.
Opteron Una versión de gama alta del Xeon Pentium 4, diseñada para ordenadores de servidor y sistemas multiprocesador
